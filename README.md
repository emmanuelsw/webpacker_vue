# Rails 5.2 + Vue.js 2 (Webpacker)

* Rails 5.2
* Vue.js 2
* Webpacker
* iView


### Setup

```zsh
git clone git@github.com:emmanuelsw/webpacker_vue.git
cd webpacker_vue
bundle install & yarn install
rails db:create
hiveman #(brew install hiveman)
```
